# ConfigReader
A lightweight, robust configuration management library for Kotlin and Java.

## Feature
* Read configuration file and parsing
* Map your config files directly to **Classes**
* Automatically creates default config files if they don't exist.

## Supported extensions

* `yml`
* `yaml`
* `properties`

... and we plan to provide more config format!
